## 📋 Sobre o Projeto

Este projeto foi desenvolvido durante o curso de **HTML5 e CSS3** do professor Gustavo Guanabara. O site aborda a história do sistema operacional Android, desde sua criação até os dias atuais.

**O que foi praticado:**
- Estruturação de páginas com HTML5 semântico
- Estilização e layout com CSS3
- Uso de fontes personalizadas e ícones
- Design responsivo
